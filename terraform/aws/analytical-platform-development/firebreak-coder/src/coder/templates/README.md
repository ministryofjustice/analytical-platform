# Templates

## Packaging

```bash
tar -cvf ${TEMPLATE_NAME}.tar -C ${TEMPLATE_NAME} .
```

## Uploading

### Using the UI

1. Go to https://coder.data-platform.moj.woffenden.dev/templates/new

1. Upload packaged template

  1. Set a name e.g. `kubernetes`

  1. Set an icon e.g. `/icon/k8s.png`
